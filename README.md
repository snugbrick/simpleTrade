# simpleTrade

基于Vault的交易插件<br>

SimpleTrade（支持1.19.4+）<br>
-
SimpleTrade是一个基于Vault的交易插件<br>

使用 -commands
-
发送交易请求<br>
`/simpleTrade to <playerName> <money>`<br>
查看交易物品<br>
`/simpleview`<br>
同意交易<br>
`/simpleaccept`<br>
拒绝交易<br>
`/simplerefuse`

构建 -maven<br>
-
`mvn package`

依赖 -depends
-
depend: [ Vault ]<br>
softdepend: [ PlaceholderAPI ]<br>
